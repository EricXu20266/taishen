# 泰深 Taishen

**二班的Eric · Vibe Coding作品** | v1.0.0 | 2025

---

针对 DeepSeek 特调的 AI 智能体桌面客户端。

支持 Skill 自定义安装、MCP 自定义安装、自定义插件扩展。

> ⚠️ **仅支持 Windows 系统。** 客户端虽可连接其他大模型，但未经过实机测试，不保证连接或使用效果。

## 核心功能
- DeepSeek 全模型支持（V3/R1/V4-Pro）
- Skill 自定义安装与扩展
- MCP 服务接入
- 插件扩展系统

## 安装
- **taishenSetup 1.0.0.exe** — Windows 安装包
- **taishen免安装.zip** — 解压即用免安装版

> 📌 **仅支持 Windows 系统**
