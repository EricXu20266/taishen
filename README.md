# 泰深 Taishen

**二班的Eric · Vibe Coding作品** | v0.2.2-rc1 | 2025

---

针对 DeepSeek 特调的 AI 智能体桌面客户端。

支持 Skill 自定义安装、MCP 自定义安装、自定义插件扩展。

> ⚠️ 客户端虽可连接其他大模型，但未经过实机测试，不保证连接或使用效果。

## 核心功能
- DeepSeek 全模型支持（V3/R1/V4-Pro）
- Skill 自定义安装与扩展
- MCP 服务接入
- 插件扩展系统

## 安装
- **泰深 Setup 0.2.2-rc1.exe** — Windows 安装包
- **泰深免安装.zip** — 解压即用免安装版
