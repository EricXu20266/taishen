## 🚀 泰深 v1.3.6 正式发布

画布系统上线，浏览器独立窗口。

### 核心功能
- DeepSeek V4 全模型支持（V4-Pro / V4-Flash）
- 多供应商支持（DeepSeek / OpenAI / Mimo 等自由切换）
- 视觉代理 Myeyes — 模型不支持多模态也能"看图"，IM 渠道全面打通
- FlexDog 动态模型引擎 — 对话中随时切换模型
- Commander 形态 — AI 自主规划、调度、验证一条龙
- HTML 高级预览器 + 元素批注系统 — 所见即所得，选中即标注
- 回收站系统 — 误删会话、Skill、子代理随时捞出，永不过期
- Skill 自定义安装与扩展（兼容 agentskills.io 开放标准，支持 subgroup 子分组）
- MCP 服务接入（多源连接，支持 Streamable HTTP / SSE / stdio）
- IM 远程接入（飞书 / QQ / 微信），全面支持图片发送
- SubAgent 子代理并行调度（v3.1 架构，独立 Skill/Memory/MCP）
- 定时任务调度器 + 全局会话搜索
- macOS 双架构正式支持（x64 + arm64）

### v1.3.6

- 重大更新，taishen's desk-泰案，画布系统正式上线！泰深现在拥有自己的流式内容工作空间。
  - 独立的画布窗口，包含六种画布场景：写作、coding、html、终端、数据分析、图片。
  - 写作画布标准流式输出，边写边呈现，代码画布语法高亮+内嵌终端，写完代码直接跑测试，你看着AI忙前忙后。
  - html画布继续保持着内置预览器的批注功能，数据分析画布拥有表格与4中图形展示。
  - 所有的画布均为双向编辑模式，即泰深和你都可以在画布上进行内容编辑，历史版本追踪可以准确的记录版本信息。
- 重大更新，内置浏览器全面升级为独立窗口。从主会话窗口中剥离出来，拥有自己的窗口和完整工具栏——代理支持、多标签管理、收藏夹原生菜单、下载防呆弹窗全部 React 化，不再模态冻结主窗口。Copy 设计按钮一键把网页存到工作区，所见即所得。
- 侧边栏折叠时显示紧凑图标模式，顶部导航+底部设置一键可达，不用展开也能操作，省出更多空间给聊天区。
- 冷启动历史预算从 100K 上调到 150K，重入历史会话时上下文更完整。
- 修复了更新时候安装到错误目录的bug。
- 修复了冷启动时模型被"工具调用=文字"示范误导、零 tool_use 干瞪眼的糟糕问题。
- 安全沙箱加固：命令执行稳定性和 PathGuard 提示体验均有提升。
- 一堆零零散散的小修复。
