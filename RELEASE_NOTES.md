## 🚀 泰深 v1.2.2 正式发布

首个正式候选发布版本，专为 DeepSeek 打造的 AI 智能助手桌面客户端。

### 核心功能
- DeepSeek 全模型支持（V3/R1/V4-Pro）
- Skill 自定义安装与扩展
- MCP 服务接入
- 插件扩展系统

### 本版本修复
- 重大更新，QQ 和微信 IM 通道正式上线。现在你可以将泰深作为QQbot或是WeChat ClawBot接入到QQ和微信了。在你的手机或是电脑端 都是可以使用QQ 和微信私聊深机器人来使用泰深了。QQ支持QQ群、QQ频道机器人，WeChat ClawBot只能私聊。如果需要接入WeChat群聊需要使用企业号配置。
- 重大更新，接入飞书、QQ、WeChat后除了基础的聊天功能，3个IM工具的文件传输功能也全面打通。现在通过飞书、QQ、微信与泰深对话时，AI 生成的文档、表格、PPT 等文件都可以要求泰深发送到你的手机。鉴于Tecent官方限制，QQ群或是频道无法发送文件，仅能发送图片，QQ或是微信与机器人私聊不受影响。飞书的私聊、群聊都支持文件传输。
- 为AI思考过程中新增旋转光弧指示器 + "AI 工作中"提示——思维链输出和工具执行期间不再像"卡死"了，现在你会觉得TA在认真工作了。
- 和泰深进行了哲学讨论，稍微调整了TA的风格，现在TA的回答会更加温和、审慎、有灵性。
- 修复了空闲超时警告反复弹出的问题。
- 再次修复了1.2.1提到的一个隐藏很深的审批弹窗 Bug。
