## 🚀 泰深 v1.2.5 正式发布

专为 DeepSeek 深度调优的 AI 智能助手桌面客户端，多供应商自由切换，千万级 token 长会话每轮仅 2 分钱。

### 核心功能
- DeepSeek V4 全模型支持（V4-Pro / V4-Flash）
- 多供应商支持（DeepSeek / OpenAI / Mimo 等自由切换）
- 视觉代理 Myeyes — 模型不支持多模态也能"看图"
- FlexDog 动态模型引擎 — 对话中随时切换模型
- HTML 高级预览器 + 元素批注系统 — 所见即所得，选中即标注
- macOS 平台正式支持
- Skill 自定义安装与扩展（兼容 agentskills.io 开放标准）
- MCP 服务接入（多源连接，支持 Streamable HTTP / SSE / stdio）
- 插件扩展系统 + 工具自主创建
- IM 远程接入（飞书 / QQ / 微信）
- SubAgent 子代理并行调度
- 定时任务调度器 + 全局会话搜索

### v1.2.5 更新亮点
- 多供应商支持体系上线 — 可添加多个 AI 供应商，在不同场景下灵活切换。视觉代理同步上线，上传图片后自动转交视觉模型处理，结果无缝返回
- FlexDog 动态模型引擎 — 对话中随时切换模型，同一段对话对比不同模型的回答效果，无需新开会话
- HTML 高级预览器 + 元素批注系统 — 打开 HTML 自动进入高级预览模式，选中页面任意元素即可标注修改意见，一键发送给 AI 改代码，修改后自动刷新
- 上下文管理引擎全面升级 — 智能语义摘要保留更完整的历史逻辑，冷启动恢复后对话连贯性大幅提升
- 修复了会话搜索卡死、文件链接刷新页面等十余项问题
