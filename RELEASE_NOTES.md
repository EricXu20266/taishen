## 🚀 泰深 v1.3.5 正式发布

内置浏览器上线，搜索全面增强，经验封装引擎就位。

### 核心功能
- DeepSeek V4 全模型支持（V4-Pro / V4-Flash）
- 多供应商支持（DeepSeek / OpenAI / Mimo 等自由切换）
- 视觉代理 Myeyes — 模型不支持多模态也能"看图"，IM 渠道全面打通
- FlexDog 动态模型引擎 — 对话中随时切换模型
- Commander 形态 — AI 自主规划、调度、验证一条龙
- HTML 高级预览器 + 元素批注系统 — 所见即所得，选中即标注
- 回收站系统 — 误删会话、Skill、子代理随时捞出，永不过期
- Skill 自定义安装与扩展（兼容 agentskills.io 开放标准，支持 subgroup 子分组）
- MCP 服务接入（多源连接，支持 Streamable HTTP / SSE / stdio）
- IM 远程接入（飞书 / QQ / 微信），全面支持图片发送
- SubAgent 子代理并行调度（v3.1 架构，独立 Skill/Memory/MCP）
- 定时任务调度器 + 全局会话搜索
- macOS 双架构正式支持（x64 + arm64）

### v1.3.5

- 重大更新，内置浏览器正式上线！泰深现在可以自己打开网页、点击按钮、填写表单、截图分析了——全部在右侧面板实时可见，你就像看着他操作一样。支持多标签页管理、前进后退导航、收藏夹、自适应缩放，还能把网页设计一键下载到工作区慢慢改。
- 重大更新，经验封装系统引擎就位。泰深开始学习你的行为模式了——这不是简单的记忆，而是从你的操作中提取指纹、识别决策偏好，以制定个性化响应策略。
- 全量搜索全面增强，搜索筛选器让你按条件过滤结果，点击任意匹配片段直接定位到对话原位置，"泰深帮你搜索"将会由agent为你搜索提炼摘要。
- 新增交互协议"愤怒信号"检测。当脏话或连续?!出现时，泰深立即停止一切操作，先处理情绪再处理任务。毕竟——一条会看脸色的AI，比一条死磕到底的AI可爱多了。
- CardDock 卡片坞新增「显示全部卡片」按钮，一键展开所有隐藏卡片。
- 添加了一个记忆验证门禁，避免多会话工作时记忆混淆。
- 修复了一些了小bug。
