## 🚀 泰深 v1.3.3-plus 正式发布

卡片坞新UI、会话继承、文件预览路由升级与多项修复。

### 核心功能
- DeepSeek V4 全模型支持（V4-Pro / V4-Flash）
- 多供应商支持（DeepSeek / OpenAI / Mimo 等自由切换）
- 视觉代理 Myeyes — 模型不支持多模态也能"看图"，IM 渠道全面打通
- FlexDog 动态模型引擎 — 对话中随时切换模型
- Commander 形态 — AI 自主规划、调度、验证一条龙
- HTML 高级预览器 + 元素批注系统 — 所见即所得，选中即标注
- 回收站系统 — 误删会话、Skill、子代理随时捞出，永不过期
- Skill 自定义安装与扩展（兼容 agentskills.io 开放标准，支持 subgroup 子分组）
- MCP 服务接入（多源连接，支持 Streamable HTTP / SSE / stdio）
- IM 远程接入（飞书 / QQ / 微信），全面支持图片发送
- SubAgent 子代理并行调度（v3.1 架构，独立 Skill/Memory/MCP）
- 定时任务调度器 + 全局会话搜索
- macOS 双架构正式支持（x64 + arm64）

### v1.3.3-plus

- UI调整：卡片坞替换了右侧面板。
  - 原右侧面板内的任务清单/预算/会话文件/使用记录/动作追踪改为六张卡片，可以自由折叠拖拽排序，以及单独显隐，每个会话独立状态互不干扰，折叠偏好自动记忆。
  - 右键卡片坞空白区可唤回隐藏卡片，每张卡片支持自动收起开关，全局按钮一键隐藏卡片坞。
- 会话继承功能上线。在会话列表右键 →「继承此会话」，弹窗确认后自动裁剪上下文并开启新会话，延续之前的对话脉络，不用再从零开始跟 AI 解释背景。
- 精细化文件预览路由。4 种预览器智能分派，文件链接点击多级降级策略，不同文件类型自动匹配最佳预览方式。
- Bing Search API Key 配置页增加保存按钮和状态反馈
- 修复了新用户首次配置微信扫码失败、以及「WeChat channel not configured」误报的问题，现在初始化配置时，扫码后 IM Gateway 自动启动。
- 修复了ChatInput 右键菜单双重弹出的问题。
- 添加了一组赛博算命的内置Skill"问卦"，别问，问就开发者在新UI开发了4天后，回滚了分支，觉得自己不行，起了一卦，又行了。
  - 六壬观局（观局观势）
  - 大衍问策（必行之事的问策）
  - 火珠问断（犹豫不决的问断）
  - 奇门布局（问你都问了，当然要知道怎么做咯）
- 彩蛋一枚
